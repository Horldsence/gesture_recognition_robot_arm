# 手势识别系统 - 机械臂控制

基于树莓派的手势识别系统，通过摄像头检测手势控制机械臂移动。

## 功能特性

- **手势识别**: 向前、向后、向左、向右、向上、向下
- **位置累加**: 根据手势方向累加坐标值
- **错误处理**: 串口报错时自动停止累加
- **实时日志**: 显示发送数据和响应状态

## 手势说明

| 手势 | 触发条件 | 坐标变化 |
|------|----------|----------|
| 向前 | 手掌变小 | Z - 10 |
| 向后 | 手掌变大 | Z + 10 |
| 向左 | 手向左移动 | X - 10 |
| 向右 | 手向右移动 | X + 10 |
| 向上 | 手向上移动 | Y + 10 |
| 向下 | 手向下移动 | Y - 10 |

## 硬件要求

- 树莓派 3B+/4/5
- Raspberry Pi Camera Module 2
- 机械臂控制器 (串口 /dev/tty0)

## 安装

```bash
# 安装依赖
pip install -r requirements.txt

# 运行
python3 main.py
```

## 串口协议

### 发送帧 (11字节)

| 字节 | 内容 |
|------|------|
| 0 | 0xAA (帧头) |
| 1-2 | X (小端int16) |
| 3-4 | Y (小端int16) |
| 5-6 | Z (小端int16) |
| 7-8 | A (小端int16) |
| 9 | 校验和 |
| 10 | 0x55 (帧尾) |

### 响应格式

- 成功: `OK (x=X,y=Y,z=Z,a=A)\r\n`
- 错误: `ERR:N\r\n`

## 坐标范围

| 参数 | 范围 | 说明 |
|------|------|------|
| X | -500 ~ 500 | 水平位置 |
| Y | -500 ~ 500 | 垂直位置 |
| Z | 0 ~ 1000 | 深度 |
| A | -180 ~ 180 | 旋转角度 |

## 操作说明

1. 点击 **开始** 启动摄像头
2. 将手放入画面中
3. 移动手控制机械臂方向
4. 出现错误时点击 **复位** 清除错误状态
